Remarks
This function is primarily intended to write COMTRADE binary data records based on synchrophasor data (see Annex H: Schema for Phasor Data 2150 Using the COMTRADE File Standard in IEEE C37.111-2010), it may be necessary to manually write records for other COMTRADE needs (e.g., non 16-bit digitals).
